# Imagens servidas na raiz do site (PWA)

- **icon.png**: copiado automaticamente no deploy a partir de `assets/images/icon.png`. O servidor serve em `/assets/images/icon.png` (manifest e referências).
- **ios-adicionar-tela-inicio.png** ou **ios-adicionar-tela-inicio.gif**: opcional. Se existir, o modal "Instalar no iPhone" exibirá esta imagem/GIF mostrando onde tocar no Safari (Compartilhar → Adicionar à Tela de Início). Recomendado: captura de tela ou GIF do Safari com seta/círculo indicando o ícone Compartilhar e o item do menu.
